User can configure the target host parameter for the datalogger:

1) IP input

2) Port input

3) Interval input

Button functions: -> selected position blink -> Button 1: position left -> Button 2: value up -> Button 3: value down -> Button 4: position right -> Button 1 && Button 2: ok
